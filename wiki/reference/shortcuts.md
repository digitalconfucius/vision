# Shortcuts

Keyboard shortcuts I always half-remember. Fill in or remove sections as you use them.

## macOS system

| Combo | Action |
|-------|--------|
| `⌘ Space` | Spotlight / Raycast |
| `⌘ Tab` | Switch app |
| `⌘ \`` | Switch window within app |
| `⌘ ⌥ Esc` | Force quit |
| `⌘ ⇧ 4` | Screenshot region |
| `⌘ ⇧ 5` | Screenshot tool |

## tmux (prefix `C-b` or whatever you've bound)

| Combo | Action |
|-------|--------|
| `prefix c` | New window |
| `prefix ,` | Rename window |
| `prefix %` | Split vertical |
| `prefix "` | Split horizontal |
| `prefix o` | Next pane |
| `prefix z` | Zoom pane |
| `prefix d` | Detach session |
| `tmux a` | Re-attach |

## git

| Combo | Action |
|-------|--------|
| `git log --oneline --graph --all` | Pretty graph |
| `git reflog` | The undo log that saves you |
| `git commit --fixup <sha>` | Prep for autosquash |
| `git rebase -i --autosquash` | Apply fixups |
| `git restore --staged <f>` | Unstage |
| `git switch -c feat` | New branch |
